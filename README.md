# Event Planning Website

## Project Overview

The Event Planning Website is a web-based platform designed to simplify the process of organizing and managing events. The system enables users to plan events, schedule activities, and maintain event-related information through an intuitive interface.

## Academic Information

- University: CMRIT
- Department: Artificial Intelligence and Machine Learning
- Project Type: Group Project
- Development Approach: Prototype Development using Rocket

## Problem Statement

Traditional event planning often involves multiple manual processes that can lead to scheduling conflicts, poor communication, and inefficient management. This project aims to provide a centralized platform for event organization.

## Objectives

- Simplify event management
- Improve scheduling efficiency
- Provide a user-friendly interface
- Enhance coordination among participants
